\section{Prérequis pour le Backend}

\begin{itemize}
\item Une installation de linux (l’explication est donnée pour ubuntu)
\item python3
\item pip
\item git
\item la version devellopeur de python3
\item un compilateur
\item mysql et/ou postgresql
\item nginx
\item uwsgi
\end{itemize}
